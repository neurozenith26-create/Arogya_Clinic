ARCHITECTURE CONNECTIVITY

Frontend: React / Next.js based public website and patient portal.

Booking Engine: Handles doctor slot scheduling and appointment locking.

Diagnostics Engine: Manages tests, health packages, and cart-based bookings.

Backend: Node.js + PostgreSQL manages all clinic operations, secure report storage, and patient access management.

Redis: Used for slot locking, queues, OTP, and notifications.

Razorpay: Payment gateway for advance booking payments.
